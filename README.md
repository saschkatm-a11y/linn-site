# Für dich, Linn ❤

Eine eigenständige, interaktive Geschenk-Website mit 200 Komplimenten, mehreren Animationsvarianten, Herzensmeilensteinen, Geheimnissen, lokaler Sammlung und Tageszeitstimmung.

## Lokal starten

```powershell
npm install
npm run dev
```

Die Seite funktioniert außerdem ohne Build-Schritt über einen beliebigen lokalen Webserver. Fortschritt, Einstellungen und gefundene Schätze werden ausschließlich im Browser in `localStorage` gespeichert.

## Klang (optional)

Die Oberfläche startet immer lautlos. Wenn lokale Audiodateien ergänzt werden sollen, können diese unter `assets/audio/` als `click.mp3`, `sparkle.mp3`, `hug.mp3` und `ambient.mp3` abgelegt werden. Fehlende Dateien werden still ignoriert.
